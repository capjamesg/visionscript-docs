### Syntax

```
Replace[]
Replace["blue"]
```

### Examples

In this example, an image is loaded, faces are detected, and the first face is replaced with a blue box. The new image is then saved to a file.

```
Load["./photo.jpg"]
Replace["blue"]
Save["./new_photo.jpg"]
```
